# See your pc battery status

This script display the current battery status of your Pc.

### Used modules
- psutil==5.7.2
- win10toast==0.9
- py-notifier==0.1

You also need the Visual C++ 14> before running the script.

to install: 
```
$ pip install -r requirements.txt
```

### How to run the script
``` bash
$ python status.py
```
